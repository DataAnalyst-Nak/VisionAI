
1- Open the project folder
2- Create a VSCode's terminal, then install all required libraries by a following command (required internet).
> pip install -r requirements.txt

3- Start the Flask app
> python app.py

Note: 
    - This project is build in Window 11;
    - Python version 3.11.1; and
    - If you wish to deploy it on any cloud platform (e.g, Render, PythonAnywhere, etc.)
    please make sure your have configured the PYTHON_ENVIRONMENT=3.11.1 or 3.10.X
